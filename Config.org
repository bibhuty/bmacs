* System Initialization
Before we can build the editor, we need to bootstrap the package management system. This ensures that whenever this configuration is loaded on a new machine, Emacs knows exactly where to download our tools.

** Package Manager Setup
We use =use-package= to safely declare and install all external packages. It keeps our configuration incredibly clean and prevents Emacs from crashing if a package fails to download.

#+begin_src emacs-lisp
  (unless package-archive-contents
    (package-refresh-contents))

  ;; Initialize use-package on non-Linux platforms
  (unless (package-installed-p 'use-package)
    (package-install 'use-package))

  (require 'use-package)
  (setq use-package-always-ensure t)
#+end_src

* The Vim Engine (Evil Mode)
Emacs is the best operating system, but Vim is the best text-editing language. This section wires up =evil-mode= to give us full modal editing capabilities.

#+begin_src emacs-lisp
  (use-package evil
    :init
    (setq evil-want-integration t)
    (setq evil-want-keybinding nil)
    (setq evil-want-C-u-scroll t)
    (setq evil-want-C-i-jump nil)
    :config
    (evil-mode 1)
    (define-key evil-insert-state-map (kbd "C-g") 'evil-normal-state)
    (define-key evil-insert-state-map (kbd "C-h") 'evil-delete-backward-char-and-join)

    ;; Use visual line motions even outside of visual-line-mode buffers
    (evil-global-set-key 'motion "j" 'evil-next-visual-line)
    (evil-global-set-key 'motion "k" 'evil-previous-visual-line)

    (evil-set-initial-state 'messages-buffer-mode 'normal)
    (evil-set-initial-state 'dashboard-mode 'normal)
    (evil-set-initial-state 'vterm-mode 'emacs))

  (use-package evil-collection
    :after evil
    :ensure t
    :config
    (evil-collection-init))

*** Auto-Tangle Configuration
Automatically extract the code from this file into =config.el= every time we save.

#+begin_src emacs-lisp
  (defun my/org-babel-tangle-config ()
    "Automatically tangle our config.org file when we save it."
    (when (string-equal (file-name-nondirectory (buffer-file-name))
                        "Config.org")
      ;; Tangle the file without asking for confirmation
      (let ((org-confirm-babel-evaluate nil))
        (org-babel-tangle))))

  ;; Add the function to the 'after-save-hook' specifically for Org mode
  (add-hook 'org-mode-hook (lambda () (add-hook 'after-save-hook #'my/org-babel-tangle-config)))
#+end_src

    (setq lsp-headerline-breadcrumb-segments '(path-up-to-project file symbols))

  (use-package lsp-mode
    :commands (lsp lsp-deferred)

  (use-package lsp-ui
    :hook (lsp-mode . lsp-ui-mode)
    :custom
    (lsp-ui-doc-position 'bottom))

  ;; 1. Load the core file explorer first
  (use-package treemacs
    :ensure t
    :bind
    ("C-c t t" . treemacs))               ;; Shortcut to toggle the sidebar

  ;; 2. Make Vim keys work inside the explorer
  (use-package treemacs-evil
    :after (treemacs evil)
    :ensure t)

  ;; 3. Finally, wire the explorer up to the Language Server
  (use-package lsp-treemacs
    :after (treemacs lsp)                 ;; Only load after both treemacs AND lsp exist
    :ensure t)

** Auto Package Update
Automatically checks for package updates every 7 days, prompts you before downloading, and cleans up old versions so your Emacs directory doesn't bloat.

      (auto-package-update-hide-results t)
      (auto-package-update-delete-old-versions t)
      :config
